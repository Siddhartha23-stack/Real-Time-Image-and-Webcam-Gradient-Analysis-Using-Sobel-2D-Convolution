# Real-Time-Image-and-Webcam-Gradient-Analysis-Using-Sobel-2D-Convolution
MATLAB project for static image + real-time webcam gradient and edge detection using Sobel (2D convolution) ‚Äì Signals &amp; Systems mini-project

This project demonstrates the initial stage of image processing by performing gradient analysis and edge detection using the Sobel operator (2D convolution) in MATLAB. It supports both static image processing and real-time webcam analysis.

 Features:

Grayscale conversion

Horizontal gradient (Gx) and Vertical gradient (Gy)

Gradient magnitude computation

Normalized gradient visualization (0‚Äì255)

Binary edge map generation using thresholding

Works with both images and live webcam feed
